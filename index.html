#About me

Thanks for your interest! Allow me to share a brief overview of my IT career.


I'm Eduardo Amarante, and I work as a Data Scientist. My journey into the world of IT began during my PhD in Applied Geophysics, where I discovered my passion for technology. This realization led me to make a career change.

In 2022, I had the opportunity to work for Rede Bahia, the largest telecommunications group in the Northeast of Brazil, as a Data Analyst. This was the first step in my career transition.

My responsibilities included data cleaning, ETL processes, creating dashboards, web scraping, and setting up data warehouses, among other tasks. However, what truly drives me is using technology to address business challenges and provide essential support in the decision-making process.

In 2023, I started working as a Data Scientist at SENAI CIMATEC, one of the largest technological centers in Brazil. My role involves working on different types of projects, such as R&D and AI-as-a-Service. Working in a dynamic, innovative environment using Agile methodology and collaborating with a cross-functional team has helped me develop essential soft skills.

I am currently working on an end-to-end AI solution in an Azure Cloud environment. As a result, we are delivering a sophisticated and reliable solution to our client, materialized in a Power BI dashboard they can trust for important decision-making.

This portfolio features various projects that reflect my skill set in Business Intelligence, Data Analytics, Data Engineering, and Web Development.

Thank you for taking the time to read this, and feel free to connect with me. Cheers, Eduardo Amarante.

#Professional Experience

  • Title: Data Scientist

    Company: SENAI CIMATEC

    Period: from December 2023 to Current

    In my current project, I am working on a Business Intelligence initiative aimed at delivering value to C-level executives. This project is designed to provide actionable insights and support financial decision-making processes. We set up four layers of a Data Lake using Azure Blob Storage, and we are using Azure Data Factory to initially load data into the raw layer. Azure Data Factory is also used to build a workflow that performs the necessary transformations, applies business rules, and loads the processed data into the subsequent layers of the Data Lake. The data source is the client company's data mart. The Data Lake is built following data engineering and data governance best practices, ensuring data lineage tracking and maintaining high data quality.

    Another pipeline has been developed to automate machine learning model training using Python, with deployment handled via Azure Container Registry. As a result, the models are stored in Azure Blob Storage and are used to make predictions that feed into a Power BI dashboard. This automated pipeline is triggered by Event Grid whenever new data arrives in the final layer of the Data Lake.

    An API, built with FastAPI, integrates the system by providing an endpoint for the Power BI dashboard. The API is also deployed via Azure Container Registry. Through the dashboard, directors and C-level executives can access forecasting results for cash flow and business KPIs to support their decision-making.

    Additionally, this project is being developed using the Agile methodology. I am also responsible for mentoring the client’s professionals to ensure effective knowledge transfer, guiding them in time series analysis, modeling and forecasting, as well as teaching best practices in data visualization and dashboard development. This is very important to ensure that the clieny can maintain the project independently in the future.

    In my first R&D project, I was responsible for analyzing infrared data for material characterization, training machine learning models, building the machine learning training pipeline, and developing dashboards. I also engaged with stakeholders in bi-weekly meetings to communicate results.
